# Job Switch Analyzer 🔄

Compare your finances across Indian cities before making a job switch. Calculates tax impact (New Regime FY 2025-26), living expenses, travel costs, and tells you the minimum salary you need.

## Features
- City-wise expense defaults (Bangalore, Noida, Mumbai, Hyderabad, etc.)
- Indian income tax calculator (New Regime with marginal relief)
- Home visit & riding trip cost comparison
- Visual charts (bar + pie)
- PDF download of full analysis
- Mobile responsive

## Run locally
```bash
npm install
npm run dev
```

## Deploy to GitHub Pages
Push to `main` branch — the included GitHub Actions workflow handles the rest.
Go to repo Settings → Pages → Source: GitHub Actions.
